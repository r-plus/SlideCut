SlideCut
========

Slide shortcut via space key.    
Alternate portion of `KBShortcuts` feature for iOS 7 and arm64 devices.

#### Usage
Tap `Space` key then slide to supported key.

* X : Cut
* C : Copy
* V : Paste
* A : Select All
* Z : Undo
* Y : Redo
* Q : Start line
* P : End line
* B : Beginning of Document
* E : End of Document
* S : Select word
* H : Move to Left
* J : Move to Down
* K : Move to Up
* L : Move to Right
