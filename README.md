SlideCut
========

Slide shortcut via space key.    
Alternate portion of `KBShortcuts` feature for iOS 7 and arm64 devices.

#### Usage
Tap `Space` key then slide to supported key.

* X : Cut
* C : Copy
* V : Paste
* A : Select All
* Z : Undo
* Q : Start line
* P : End line
* H : Home
* E : End
* S : Select word
